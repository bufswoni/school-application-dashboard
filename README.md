# 고교별 지원/등록 현황 대시보드

2024~2026학년도 3개년 고등학교별 지원/등록 인원 분석 대시보드입니다.

## 주요 기능

- **KPI 요약**: 2026 총 지원·등록 인원, 등록률, 3개년 누적 현황
- **상위 30개교 현황표**: 연도별 지원·등록 인원, 증감 표시, 컬럼 정렬
- **필터**: 지역·고교구분 필터 및 고교명 검색
- **차트**:
  - 상위 10개교 3개년 지원인원 추이 (Line)
  - 상위 10개교 등록인원 비교 (Bar)
  - 고교유형별 분포 (Doughnut)
  - 25→26 지원 증감 Top/Bottom 10 (Horizontal Bar)

## 데이터

- 대상: 3개년(2024~2026) 지원인원 합계 기준 상위 30개 고등학교
- 지역: 부산광역시, 경상남도, 울산광역시 등
